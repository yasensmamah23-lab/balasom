# منصة بلسم للرعاية الصحية المنزلية (Balsam Platform)

منصة رقمية متكاملة تهدف إلى تقديم خدمات الرعاية الصحية المنزلية في الأردن بجودة عالية واحترافية.

## الهدف من المشروع
تسهيل وصول المرضى لكادر تمريضي خبير ومؤهل، وضمان تقديم الرعاية اللازمة في مكان تواجد المريض بكل سهولة وأمان.

## المميزات الرئيسية
* **خدمات طبية شاملة:** تشمل غيار الجروح، العناية بمرضى السكري، والمتابعات الحيوية.
* **تجربة مستخدم سهلة:** واجهات مصممة خصيصاً للمستخدمين وللكادر التمريضي لضمان سرعة الوصول للخدمة.
* **نظام تقييم:** تعزيز الثقة والجودة من خلال تقييمات المرضى للخدمات المقدمة.

## التقنيات المستخدمة
* **Frontend:** Tailwind CSS
* **Structure:** HTML5
