# منصة بلسم (Balsam) للرعاية الصحية المنزلية

بلسم هي منصة تقنية رائدة تهدف إلى ربط الكوادر التمريضية المحترفة بالمرضى في كافة محافظات المملكة (عمان، الزرقاء، وغيرها)، لتقديم خدمات رعاية منزلية آمنة، سريعة، وذات جودة عالية.

## مميزاتنا:
* **نطاق خدمة واسع:** نصلكم أينما كنتم في المملكة؛ نقيم المريض ويتم الاتفاق في مكانكم، فلا داعي لأن تتعبوا أنفسكم.
* **نظام تسجيل مخصص:** بوابات دخول منفصلة للممرضين والمستخدمين لضمان خصوصية وتنظيم البيانات.
* **لوحة تحكم ذكية:** تتيح للممرضين إدارة حالاتهم واستقبال طلبات الزيارات فوراً.
* **سجل المريض السريع:** يوفر للممرضين تفاصيل الحالة الصحية للمريض لضمان الاستعداد المهني التام.
* **نظام طوارئ (SOS):** زر مخصص للمرضى لطلب المساعدة الفورية في الحالات الحرجة.
* **تقييم جودة الخدمة:** نظام تقييم بعد كل زيارة لضمان أعلى معايير الأداء والاحترافية.

## رؤيتنا:
الارتقاء بقطاع الرعاية الصحية المنزلية عبر حلول رقمية مبتكرة، تضع راحة المريض واحترافية التمريض في صدارة أولوياتها.
