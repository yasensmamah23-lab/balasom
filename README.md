# منصة بلسم (Balsam) للرعاية الصحية المنزلية

بلسم هي منصة تقنية متخصصة تهدف إلى ربط الكوادر التمريضية المحترفة بالمرضى في مدينة الزرقاء، الأردن، لتقديم خدمات رعاية منزلية آمنة، سريعة، وذات جودة عالية.

## مميزات المنصة:
* **نظام تسجيل مخصص:** بوابات دخول منفصلة للممرضين والمستخدمين لضمان خصوصية وتنظيم البيانات.
* **لوحة تحكم ذكية للممرض:** تتيح للممرضين إدارة حالاتهم (متاح، مشغول، خارج الخدمة) واستقبال طلبات الزيارات فوراً.
* **سجل المريض السريع:** يوفر للممرضين تفاصيل الحالة الصحية للمريض قبل قبول الطلب لضمان الاستعداد المهني.
* **نظام طوارئ (SOS):** زر مخصص للمرضى لطلب المساعدة الفورية في الحالات الحرجة.
* **تقييم جودة الخدمة:** نظام تقييم بعد كل زيارة لضمان أعلى معايير الأداء والاحترافية.

## الهدف:
تطوير قطاع الرعاية المنزلية في الزرقاء عبر تسخير الحلول الرقمية لتوفير وقت الممرضين وتحسين وصول المرضى للرعاية الطبية.
